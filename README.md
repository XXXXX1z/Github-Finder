# GitHub Finder

A small React app that searches GitHub users and displays basic profile information.

## Features

- Search for a GitHub user by username
- Display user login, ID, profile URL, name, followers, and following count
- Fetch data from a local backend API

## Tech Stack

- React
- JavaScript
- CSS
- Node.js / Express

## Getting Started

Install dependencies:

```bash
npm install

Start the backend server:

npm run server

Start the frontend:

npm run dev

Open the app in the browser and search for a GitHub username.

## API

The frontend calls:

GET http://localhost:3000/api/github/:username

Example:

GET http://localhost:3000/api/github/octocat

```
